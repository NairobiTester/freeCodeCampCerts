Introduction

Python is a popular high-level programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python emphasizes code readability with its clean syntax and uses indentation instead of braces or keywords like other programming languages.

With Python, developers can build a wide range of applications, including web development, scientific computing, data analysis, machine learning, and automation. Its extensive standard library and vast ecosystem of third-party packages make it a versatile language for various domains and projects.

  • Python is widely used in data science and machine learning due to libraries like NumPy, Pandas, and TensorFlow.
  • Python has a large and active community that provides abundant resources, tutorials, and support.
Python Basic Syntax

Python uses a straightforward syntax that makes it easy to understand and write code. Here are some key elements of Python's basic syntax:

  • Indentation: Python uses indentation (typically four spaces) to define blocks of code instead of curly braces or keywords like "end". This promotes readability and enforces code consistency.
  • Variable assignment: Variables in Python are assigned using the equal sign (=). For example, name = "John" assigns the string "John" to the variable "name".
  • Print statements: The print() function is used to display output in Python. For example, print("Hello, World!") prints the string "Hello, World!" to the console.
  • Comments: Comments in Python start with the hash symbol (#). They are used to explain code and make it more understandable for others.
Comments

Comments in Python are used to provide explanations or annotations within the code. They are not executed by the interpreter and do not affect the program's functionality. Here are the different types of comments in Python:

  • Single-line comments: # This is a single-line comment.
  • Multi-line comments: ''' This is a multi-line comment. It can span across multiple lines. '''
  • Documentation strings (docstrings): def my_function(): """ This is a docstring. It provides a description of the function. """ # Function code here
Variables

In Python, variables are used to store data values. Here are a few points about Python variables:

  • Variable assignment: Variables in Python are assigned using the equal sign (=). For example, name = "John" assigns the string "John" to the variable "name".
  • Dynamic typing: Python is dynamically typed, meaning the type of a variable is determined at runtime. You can assign different types of values to the same variable.
  • Variable naming: Variable names should follow certain rules. They must start with a letter or underscore, and can contain letters, digits, and underscores. They are case-sensitive.
  • Variable reassignment: You can reassign a variable to a new value at any point in your code. The variable will then reference the new value.

Here are some examples of Python variable usage:

# Assigning values to variables name = "John" age = 25 is_student = True # Reassigning a variable age = 26 # Printing variable values print(name) print(age) print(is_student)
Data Types

Python provides several built-in data types to store and manipulate different kinds of information. Here are some important Python data types:

- Numbers: Python supports integers, floating-point numbers, and complex numbers. Example: 42, 3.14, 1+2j

- Strings: Strings are sequences of characters and are used to represent text in Python. Example: "Hello, World!"

- Lists: Lists are ordered collections of items that can be of different data types. They are mutable and allow duplicate values. Example: [1, 2, 'apple', True]

- Tuples: Tuples are similar to lists but are immutable, meaning their elements cannot be modified once defined. Example: (1, 2, 'banana')

- Dictionaries: Dictionaries are key-value pairs that allow you to store and retrieve values based on a unique key. Example: {"name": "John", "age": 25}

- Sets: Sets are unordered collections of unique elements. They support operations like union, intersection, and difference. Example: {"apple", "banana", "orange"}

Operators

Python provides a variety of operators to perform different operations on variables and values. Here are some important Python operators:

  • Arithmetic operators: These operators perform mathematical operations on numeric values, such as addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).
  • Assignment operators: These operators are used to assign values to variables, such as the equal sign (=), plus-equal (+=), minus-equal (-=), and more.
  • Comparison operators: These operators compare two values and return a Boolean result, such as equal to (==), not equal to (!=), greater than (>), less than (<), and more.
  • Logical operators: These operators are used to combine conditional statements, such as and (and), or (or), and not (not).
  • Bitwise operators: These operators perform operations on binary representations of numbers, such as bitwise AND (&), bitwise OR (|), bitwise XOR (^), and more.

Here are some examples of Python operators in action:

# Arithmetic operators a = 5 + 3 b = 10 / 2 # Assignment operators x = 7 x += 3 # Comparison operators age = 25 is_adult = age >= 18 # Logical operators is_sunny = True is_warm = True is_summer = is_sunny and is_warm # Bitwise operators x = 5 & 3 y = 10 | 2
Decision Making

Decision making is a fundamental aspect of programming. In Python, decision making is typically achieved using conditional statements, such as the if, elif, and else statements. These statements allow you to execute different blocks of code based on certain conditions.

Here is an example that demonstrates decision making in Python:

# Example 1 num = 10 if num > 0: print("The number is positive.") elif num < 0: print("The number is negative.") else: print("The number is zero.") # Example 2 age = 25 if age >= 18: print("You are an adult.") else: print("You are not an adult yet.")

In the first example, the code checks if the variable num is positive, negative, or zero, and prints the corresponding message. In the second example, the code checks if the variable age is greater than or equal to 18 and prints the appropriate message based on the condition.

Decision making allows you to control the flow of your program and execute different actions based on specific conditions. It is an essential concept in programming to handle different scenarios and make your code more flexible and intelligent.

Loops

Loops are used in Python to repeatedly execute a block of code. There are two types of loops in Python: for loops and while loops.

Here is an example of a for loop that iterates over a list of numbers:

numbers = [1, 2, 3, 4, 5] for num in numbers: print(num)

Strings

Lists

Lists are ordered collections of items in Python. They can contain elements of different data types and are mutable, meaning their elements can be modified.

Here is an example of creating and accessing a list:

fruits = ['apple', 'banana', 'orange'] print(fruits[0])
Tuples

Tuples are similar to lists but are immutable, meaning their elements cannot be modified once defined. They are often used to store related pieces of information together.

Here is an example of creating and accessing a tuple:

person = ('John', 25, 'USA') print(person[1])
Dictionary

Dictionaries are key-value pairs in Python. They allow you to store and retrieve values based on a unique key. Dictionaries are mutable and unordered.

Here is an example of creating and accessing a dictionary:

student = {'name': 'John', 'age': 25, 'country': 'USA'} print(student['age'])
Functions

Functions in Python are reusable blocks of code that perform specific tasks. They allow you to break your code into smaller, organized units and promote code reusability.

Here is an example of defining and calling a function:
